# 📍 기능 목록

### 게임 진행

- 1부터 9까지의 숫자 중 서로 다른 3개를 골라 정답 생성
- 사용자로부터 숫자 3개 입력 받기
  - 입력값이 3자리가 아닌 경우 예외 발생
  - 입력값이 숫자가 아닌 값을 포함하는 경우 예외 발생
  - 입력값에 중복되는 숫자가 있을 경우 예외 발생
- 입력 받은 숫자와 정답 비교 후 힌트 출력
  - 스트라이크 개수 세기
  - 볼 개수 세기
- 정답일 경우 정답 메세지 출력 후 게임 종료

### 게임 종료 후

- 게임 재시작 여부 묻고 사용자 입력 받기
  - 1 입력한 경우 게임 재시작
  - 2 입력한 경우 게임 완전히 종료
